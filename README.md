# Modell-vasut-sorompo-vezerlo
A projekt célja egy Arduino alapú modell vasút sorompó vezérlő megvalósítása, amely képes 2db optikai szenzorral meghatározni a vonat közeledését és távozását az átjáróból.
Modell vasút sorompó vezérlő  - IOT/Digitális Technika 2. Beadandó

Ez a projekt a Neumann János Egyetem IOT/Digitális Technika 2. tantárgyának beadandó feladata.
